# Skills Repo

This repo showcases custom Claude Code skills. Each skill lives in `skills/<skill-name>/SKILL.md`.

## Keeping the README up to date

Whenever a skill is added to or removed from the `skills/` folder, update `README.md` to match — add or remove the corresponding section under **Skills**, following the existing format: skill name as a heading, a one-line pitch in bold, a short description, and a link to the `SKILL.md` file.
